Basic usage example of contao-rpc

For demonstration we have setup this basic CRUD application with a sign-in process. As you see this GUI is completely encapsulated from the Contao installation. The GUI is hosted on Github and Contao is running on a separate Server. The only point of contact between GUI and Contao is through contao-rpc.

Sign-In can be proceeded with any Frontend-User of the "Musikakademie" Theme. To get a better understanding how the GUI communicates with contao it's recommended that you open up the Devtools of your browser and inspect the Ajax Requests within the network tab.

Checkout the Server-Side code of the application here. The GUI is built with Backbone.JS, jQuery and Bootstrap and can be viewed here.
